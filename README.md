# Flexbox-Grid

Este proyecto es una demostración visual de todas las propiedades principales de Flexbox y CSS Grid.
Está diseñado con fines educativos para entender cómo funcionan las propiedades de CSS.

FLEXBOX

Se incluyen las siguientes propiedades:

*Dirección (flex-direction)

- "row"
- "row-reverse"
- "column"
- "column-reverse"

* Alineación eje principal (justify-content)

- "flex-start"
- "flex-end"
- "center"
- "space-between"
- "space-around"
- "space-evenly"

* Alineación eje cruzado (align-items)

- "stretch"
- "flex-start"
- "flex-end"
- "center"
- "baseline"

* Control de líneas (flex-wrap)

- "nowrap"
- "wrap"
- "wrap-reverse"

 * Orden y flexibilidad

- "order"
- "flex-grow"
- "flex-shrink"
- "flex-basis"

* Ajustes específicos

- "align-self"
- "align-content"

* Espaciado

- "gap"

---

*CSS GRID*

Se incluyen las siguientes propiedades:

* Estructura

- "grid-template-columns"
- "grid-template-rows"

* Unidad flexible

- "fr"

* Espaciado

- "gap"

* Posicionamiento

- "grid-column"
- "grid-row"

* Funciones

- "repeat()"

* Responsive

- "auto-fit"
- "auto-fill"

* Alineación

- "align-items"
- "justify-content"

--------------------------------

Objetivo

- Comprender cómo funciona Flexbox en una dimensión
- Comprender cómo funciona Grid en dos dimensiones
- Visualizar el comportamiento de cada propiedad
- Servir como material de estudio o apoyo académico

---

- Flexbox → Trabaja en una sola dirección (fila o columna)
- Grid → Trabaja en filas y columnas al mismo tiempo
